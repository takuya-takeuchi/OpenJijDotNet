# Tutorial
 
This program is ported by C# from src/OpenJij/project_template/template.cpp.

## How to use?

## 1. Build

1. Open command prompt and change to &lt;Tutorial_dir&gt;
1. Type the following command
````
dotnet build -c Release
````
3. Copy ***OpenJijDotNet.dll*** and ***OpenJijDotNetNative.dll*** to output directory; &lt;Tutorial_dir&gt;\bin\Release\netcoreapp3.1.

## 2. Run

````
cd <Tutorial_dir>
dotnet run --configuration Release
````
